Jarvis 人生系統 Starter

這是一份給 AI 新手的安裝指引。先選你的電腦版本，下載乾淨 starter kit，解壓縮後用 Codex 打開，就能開始使用記憶架構、協作模式、Harness 規範，以及製作 Agent / Skill 的能力模組。

2 個版本Mac 與 Windows 分開下載。
0 私人資料只有空白架構、範本與教學。
5 個 Skills含新手導覽與 Agent/Skill Builder。

Mac 版下載

照這 5 步走

這不是工程師專用文件。你只要下載、解壓縮、用 Codex 打開資料夾，然後貼上第一句話。

01

下載對應版本

在上方選 Mac 或 Windows，優先使用 Google Drive 下載。若 Drive 無法使用，可改用 GitHub 備用下載。

02

解壓縮資料夾

把 zip 解壓縮到 Documents、桌面或你熟悉的位置。Windows 使用者請不要直接在壓縮檔內開啟。

03

打開 Codex App

安裝並登入 Codex App，選擇 Local 模式，再開啟剛剛解壓縮的 Jarvis 資料夾。

Codex App 官方說明
04

貼上第一句話

讓 Jarvis 用新手導覽 Skill 帶你建立第一個任務。

我完全不懂 AI，請用 $new-user-onboarding 帶我完成設定，並幫我建立第一個任務。
05

確認成功

如果 Jarvis 能說明資料夾結構，並建立第一個 session-state 或下一步清單，就代表安裝完成。

請檢查這個 starter kit 是否安裝成功，並告訴我下一步該做什麼。

包裡有什麼

只留下幾乎所有人都用得到的核心，不放個人知識庫、不放私人專案。

Memory空白記憶架構

`session-state`、`handoffs`、`context-window`，用來讓任務下次接得回來。

Collab協作模式

理解、拆解、執行、驗證、交接，讓新手知道 AI 正在做哪一步。

HarnessProof Contract

要求每次完成都能留下可檢查證據，避免看似完成其實沒有落地。

Agent新手導覽員

任何人第一次打開，都能用 `$new-user-onboarding` 開始。

SkillAgent / Skill Builder

協助把重複任務變成 Agent、Skill 或固定 Workflow。

Templates可改造範本

包含 Agent YAML、Skill Markdown、任務狀態與驗收清單。

Privacy by default

這份包不含個人資料

  • 沒有私人 vault、客戶資料、商業策略或對話紀錄。
  • 沒有 API key、token、帳號設定或本機同步設定。
  • 所有記憶檔都是空白骨架，使用者自己決定要保存什麼。
  • 公開頁與 zip 可從 GitHub 檢查內容。
